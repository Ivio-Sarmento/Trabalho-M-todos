# Efeito do MEI na Formalização de Empresas no Brasil

## Objetivo

Investigar se a redução da carga tributária proporcionada pelo Microempreendedor Individual (MEI), criado em 2009, está associada ao aumento da formalização de empresas em setores com alta incidência de informalidade.

## Base de Dados

* SIDRA/IBGE – Tabela 992 (Cadastro Central de Empresas - CEMPRE)
* Período: 2006–2021

## Setores Analisados

* Comércio e reparação de veículos automotores
* Alojamento e alimentação
* Reparação e manutenção de equipamentos e objetos pessoais

## Ferramentas Utilizadas

* Python
* Pandas
* Requests
* Matplotlib

## Estrutura do Projeto

* `Trabalho.py`: código principal
* `Dados/`: bases utilizadas
* `Gráficos/`: gráficos gerados
* `requirements.txt`: dependências do projeto

## Pergunta de Pesquisa

A redução dos impostos promovida pela criação do MEI contribuiu para reduzir a informalidade ao incentivar a formalização de empresas?
